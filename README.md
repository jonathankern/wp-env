# Prerequisites

+ [Vagrant](http://www.vagrantup.com/downloads.html)
+ [Virtualbox](https://www.virtualbox.org/wiki/Downloads)

path to wp

## Getting Started

1. Clone the project.
2. Run the command `vagrant up` from the directory
